# FoodStack - An AI Enabled Smart Food Finder System

**FoodStack (formerly LocalConnect)** is a hybrid mobile application that seamlessly integrates social media engagement with food discovery and ordering services. The platform enables users to create and share short video reels showcasing food, lifestyle, and local experiences while simultaneously exploring nearby restaurants and food stalls for ordering or booking. 

By combining entertainment, community interaction, and utility in a single ecosystem, the app strengthens connections between users and local businesses. 

## Features
- AI-driven personalized recommendations
- Real-time menu updates
- Digital payment integration
- Loyalty rewards
- Geo-location services
- Multi-language support
- Role-based access (Customer, Restaurant, Admin)
- Reel-based food discovery

## Getting Started

1. Make sure you have Flutter installed.
2. Open a terminal in this directory.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to start the app.
